# FoodSaver

FoodSaver is an Excess Food Donation Platform designed to reduce food wastage by connecting donors with NGOs and volunteers.

## Features

- Donor Dashboard
- NGO Dashboard
- Volunteer Dashboard
- Food Listings
- Claim Requests
- Pickup Tracking
- Donation History
- Dashboard Analytics
